Toy Rebellion Unity Unit Sprites

Each attack sheet:
- 6 frames
- 256 x 256 pixels per frame
- 1536 x 256 pixels per sheet
- Transparent PNG
- Facing right
- Fixed baseline and consistent scale

Unity:
1. Import the three *_attack_sheet.png files.
2. Texture Type: Sprite (2D and UI)
3. Sprite Mode: Multiple
4. Filter Mode: Bilinear
5. Compression: None
6. Open Sprite Editor.
7. Slice > Grid by Cell Size > 256 x 256.
8. Pivot: Bottom Center.
9. Apply.
10. Select all six slices and create an Animation.
11. Use 8–10 samples per second and disable Loop Time.

Damage timing:
Action Figure: frames 2–3
Robot Toy: frames 3–4
Plushie: frames 3–4
